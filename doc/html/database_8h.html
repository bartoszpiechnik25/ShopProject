Detailed Description

This file contains the implementation of the ShopDatabase class. The ShopDatabase class provides an interface for performing database operations on books and phones. The class uses std::map and std::vector to store the data internally and provides various methods for adding, deleting, and modifying the data. The class also provides a mechanism for loading and saving the data from and to files.
